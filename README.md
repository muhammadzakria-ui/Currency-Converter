Currency Converter
A beautiful, real-time currency converter that works in your browser!
Convert any amount from one currency to another with live exchange rates, country flags, and a smooth dark/light mode toggle.

✨ Main Features

Real-time Exchange Rates – Powered by a free API
190+ Currencies – Choose from tons of countries
Country Flags – See flags next to each currency
Swap Currencies – Click the arrow to switch "From" and "To"
Dark & Light Mode – Toggle with the sun/moon button
Responsive Design – Works perfectly on mobile, tablet, and desktop
Smart Input – Automatically fixes invalid amounts


How to Use (Step-by-Step)

Enter an amount (default is 1)
Pick a currency from the "From" dropdown
Pick a currency from the "To" dropdown
Click "Get Exchange Rate"
See the result instantly! Example:1 USD = 286.1234 PKR
Want to swap? Click the swap icon
Toggle dark mode using the sun/moon button in the top-right

How to Run the Project
Option 1: Just Open in Browser (Easiest!)

Save these 4 files in the same folder:
index.html
style-res.css
code.js
script.js

Double-click index.html → opens in your browser!
Start converting currencies!

Note: You need internet to fetch live rates.

Option 2: Use Live Server (For Developers)

Open the folder in VS Code
Install Live Server extension
Right-click index.html → "Open with Live Server"
Auto-reload on changes!
